# Time Stopper

5月21日（水）　完成目標

**目指せ68** は、タイマーを使って「タイムストップ」ゲームを楽しむためのシンプルなアプリケーションです。  
ユーザーは、タイマーが進んでいく中で「STOP」ボタンを押して、どれだけ正確に時間を止められるかを競います。
